Using WMU's VPN

Group Members: Marcel Englmaier, Thinh Nguyen, David Rice

Here's how to get WMU's vpn up and running

Download Go to gowmu.wmich.edu
Log in using your BroncoNetID
Scroll down a bit. On the left side, you'll see a "Software Downloads" box
Inside that box, you'll see "VPN - Virtual Private Network Downloads"
The page will reload, so you have to scroll back down to it
Find "Network Connect for 64-bit Windows"
Click the download link and save the file

Installation Run the file you just downloaded

Using Network Connect Run the "Network Connect" Program
Tt'll give you a text box or line where you type in "vpn.wmich.edu" and click Go
Use your BroncoNetID on the page given to you to log int
A few pop-up will pop up. Click yes
Now you should get a window with statistics and information, including a Session Timer
You are done and in the vpn.
To log out, click "Sign Out"
To Reconnect, simply repeat the above steps starting with "Run "Network Connect""
