Using Apache

Group Members: Marcel Englmaier, Thinh Nguyen, David Rice

The following is a guide to installing Apache, setting up the firewall, setting Apache to start with your server, installing php, and where to put the main website files once you've done that.
Each line is a line you need to enter into your shell. You can directly copy each line and it should work.
Lines beginning with a "#" can be ignored for your shell. These are only comments detailing what is going to happen

Installing Apache

sudo yum install httpd mod_ssl
sudo /usr/sbin/apachect1 start
sudo nano /etc/httpd/conf/httpd.conf
#now find the '#ServerName www.example.com:80' line and change it to 'ServerName [servername that you want]'
sudo /usr/sbin/apachect1 restart

Setting Up the Firewall

sudo iptables -I INPUT -p tcp --dport 80 -j ACCEPT
sudo service iptables save
#now you can go to your server's IP or Domain and get the default Apache splash screen

Setting Apache to Automatically Start on System Power-On

sudo /sbin/chkconfig httpd on
#now check that it worked
sudo /sbin/chkconfig --list httpd
# response:httpd 0:off 1:off 2:on 3:on 4:on 5:on 6:off

Installing PHP5

#PHP5 install
sudo yum install php-mysql php-devel php-gd php-pecl-memcache php-pspell php-snmp php-xmlrpc php-xml
#restart apache
sudo /usr/sbin/apachect1 restart

Adding your First Webpage

#Navigate to Apache's webpage directory cd /var/www/html
#make a basic page
sudo nano index.html
#now type in basic info
#hit ^X(ctrl+X), then 'y' to save #now you can go to your server's IP or Domain and get the page you just entered
