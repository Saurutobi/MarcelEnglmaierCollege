Basic Package Management With Yum

Installing New Software

To install a package 'mypackage', enter the command:
sudo yum install mypackage

To install a package group 'MyGroupPackage', enter the command:
sudo yum groupinstall "MyGroupPackage"

Updating Software

To update a package 'mypackage' to the latest version, enter the command:
sudo yum update mypackage

To update all of the packages in a package group 'MyGroupPackage', enter the command:
sudo yum groupupdate "MyGroupPackage"

Removing Software

When removing software, yum checks your system for the specified software and its dependencies.
Both the specified software and the dependencies will be removed

To remove a package 'mypackage' from your system, enter the command:
sudo yum remove mypackage

To remove all of the packages in a package group 'MyGroupPackage', enter the command:
sudo yum groupremove "MyGroupPackage"

Searching for Packages

The search feature of yum returns software available from the configured repositories or already installed on your system.

To search for specific package by name, use the 'list' function. To search for package 'mypackage', use command:
sudo yum list mypackage

To search for specific versions of a package, say version 0.123, use the command:
sudo yum list mypackage-0.123

Advanced Searches

When the name of a specific package is unknown, the 'search' and 'provides' commands can be used.

The 'search' command checks the names, descriptions, summaries and listed package maintainers.
To search for all packages that relate to 'PalmPilot', use the command:
sudo yum search PalmPilot

The 'provides' command checks both the files included in the packages and the functions the software provides.
To search for all packages that include files called 'libneon', use the command:
sudo yum provides libneon

When searching with partial word or name, use the escape character '\' followed by '?' to represent one character or '*' to mean zero or more characters.
To 'list' all packages with names that begin with 'tsc', use the command:
sudo yum list tsc\*
